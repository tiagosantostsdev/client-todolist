API TODOLIST 

Está é o front-end desenvolvida para a api da lista de tarefas.

TECNÓLOGIAS

1. react
2. ReactJs

INSTALAÇÃO
Passos para instalar o projeto:

git clone https://github.com/tiagosantostsdev/client-todolist
cd repositorio
npm install

USO
Como rodar o projeto:

npm run dev
